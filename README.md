# Birthday App

## Installation
```bash
$ npm install
```
## Setup
- Create database with name <code>birthday_notifier</code>.
- Setup <code>.env</code> with your db properties.

## Running the app
```bash
$ npm run dev
```
OR
```bash
$ npm start
```

- <code>POST /api/user</code>
